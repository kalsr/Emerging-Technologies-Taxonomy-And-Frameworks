AI-TAXONOMY-FRAMEWORK

This platform explains Artificial Intelligence through a structured taxonomy and visual framework. It organizes AI into three major domains—Learning & Intelligence, Advanced Computing, and AI Systems—and allows users to explore each category through a sidebar‑driven interface. The app uses Graphviz to generate dynamic diagrams that visually illustrate AI pipelines, neural networks, NLP flows, quantum models, and more. Each subcategory includes real‑world examples, explanations, and simplified architecture diagrams to make complex AI concepts easy to understand. The code also introduces a high‑level AI structure diagram that maps how major AI domains relate to each other. Streamlit powers the UI, enabling responsive layouts, dropdown menus, and interactive content sections. This framework helps students, engineers, and business leaders understand how AI technologies fit together, how data flows through AI systems, and how different AI disciplines support real‑world applications. By combining taxonomy (classification) with framework (architecture), the app becomes a practical learning tool for training, presentations, and strategic planning. It ultimately transforms AI into a visual, intuitive, and highly accessible experience.

Critical Infrastructure Security Framework

The Critical Infrastructure Security Framework Analyzer is an interactive Streamlit-based cybersecurity application that helps organizations assess, monitor, and improve the security of critical infrastructure systems using the NIST Cybersecurity Framework. The platform organizes 21 security control areas across the five NIST pillars—Identify, Protect, Detect, Respond, and Recover—providing detailed explanations, implementation guidance, and references to standards such as NIST SP 800-53, ISO/IEC 27001, COBIT, and ISA/IEC 62443. Key features include interactive navigation, Mermaid and Graphviz security diagrams, synthetic assessment data generation, implementation status tracking, and exportable reports in PDF, Word, and JSON formats. The application is designed for security analysts, auditors, planners, and administrators in government, defense, utilities, healthcare, transportation, and other critical infrastructure sectors, helping them identify security gaps, support compliance efforts, improve incident response readiness, and enhance organizational resilience against cyber threats.

Edge‑Computing Taxonomy & Framework

Edge‑Computing Taxonomy & Framework provides a unified, visual, and interactive way to understand how cloud, edge, and IoT systems work together in real‑world environments. It organizes the entire ecosystem into functional categories, use cases, and architectural challenges, then visualizes each through Concept, Flow, Data‑Movement, Control‑Plane, and Edge–Cloud Interaction diagrams. Built using Python, Streamlit, Graphviz, and JSON/Doc/PDF export tools, the framework behaves like a live reference model that engineers, architects, and decision‑makers can explore. Each module shows how data travels, how control signals flow, and how security boundaries are enforced across devices, gateways, and cloud platforms. The system mirrors real‑world edge deployments such as smart homes, smart cities, industrial IoT, and collaborative edge clusters. By processing data closer to the source, the framework highlights benefits like reduced latency, lower bandwidth usage, improved privacy, and faster decision‑making. It also exposes operational challenges—programmability, naming, data abstraction, service management, and security—helping teams design more resilient architectures. Overall, this application acts as a complete educational, architectural, and decision‑support tool for anyone building or evaluating modern edge‑computing solutions.

Email Isolation & Threat Control Framework

The Email Isolation & Threat Control framework is an interactive cybersecurity and email security analysis application designed to help organizations identify, isolate, monitor, and analyze potentially malicious emails before they reach end users. The platform uses Python Streamlit for the user interface, Pandas and NumPy for data processing, Plotly for interactive dashboards and visual analytics, and ReportLab and Python-Docx for professional report generation. Key features include email threat categorization (Phishing, Spam, Malware, Business Email Compromise, and Safe Emails), risk scoring and confidence analysis, real-time dashboard metrics, interactive pie charts and risk analytics, synthetic email dataset generation, and CSV/JSON file upload capabilities for analyzing real-world email data. The framework provides security teams with visibility into high-risk email threats, enabling faster identification of phishing campaigns, malware delivery attempts, and business email compromise attacks. Users can explore detailed datasets, analyze email risk distributions, and generate executive reports through PDF, Word, and JSON export functions. Real-world benefits include improved email security posture, reduced risk of successful cyberattacks, enhanced threat awareness, support for Security Operations Center (SOC) monitoring, faster incident investigation, cybersecurity training and simulation exercises, and better protection of organizational users, data, and critical business operations from evolving email-based threats.

IoT Sensor Data File Management Framework

The IoT Sensor Data File Management Application Framework is a comprehensive Internet of Things (IoT) data management and administration framework designed to help organizations efficiently manage, organize, and maintain sensor-generated data files used in machine learning, artificial intelligence, data analytics, and data mining environments. Built using Python Streamlit, the platform leverages JSON, ReportLab, Python-Docx, Mermaid Diagrams, Graphviz Visualizations, and Interactive Data Tables-- to provide a user-friendly interface for managing IoT datasets. Key features include creating new IoT sensor data files, creating files within specific directories, reading and validating existing sensor datasets, deleting duplicate files, determining file sizes for storage and capacity planning, copying files for backup and version control, tracking file modification dates for auditing, and appending new sensor records to existing files. The framework also generates synthetic file metadata records for demonstrations, testing, and training purposes while providing visual workflow diagrams and process flow charts to illustrate file management operations. Additionally, users can generate professional reports and export results in PDF, Microsoft Word, and JSON formats. Real-world benefits include improved IoT data governance, better storage management, enhanced data lifecycle control, support for machine learning and WEKA-based analytics, simplified audit and compliance reporting, reduced data duplication, improved operational efficiency, and more effective management of large-scale sensor data environments across smart cities, critical infrastructure, defense, healthcare, manufacturing, transportation, and industrial IoT ecosystems.
